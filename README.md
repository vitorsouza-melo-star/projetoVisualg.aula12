# Atividade 01 - Condicionais

## Exercícios de Algoritmos
### Lista 1 — Variáveis, Entrada e Saída de Dados

Bem-vindos, alunos! Nesta atividade, vocês vão praticar conceitos básicos de algoritmos com exercícios sobre entrada, saída e variáveis.

### Exercícios

1. **Algoritmo 01**
   - Imprima a mensagem:
   > "É PRECISO FAZER TODOS OS ALGORITMOS PARA APRENDER"

2. **Algoritmo 02**
   - Crie um algoritmo que imprima a média aritmética entre os números 8, 9 e 7.

3. **Algoritmo 03**
   - Crie um algoritmo que leia um número inteiro e imprima o número.

4. **Algoritmo 04**
   - Crie um algoritmo que leia dois números inteiros e imprima os dois números.

5. **Algoritmo 05**
   - Crie um algoritmo que leia um número inteiro e imprima seu sucessor e seu antecessor.

6. **Algoritmo 06**
   - Crie um algoritmo para ler um nome, endereço e telefone, e imprimi-los.

7. **Algoritmo 07**
   - Crie um algoritmo para ler dois números inteiros e imprimir a soma. Antes do resultado, deverá aparecer a mensagem: **Soma**.

8. **Algoritmo 08**
   - Crie um algoritmo para ler um número real e imprimir a terça parte deste número.

9. **Algoritmo 09**
   - Entre com dois números reais e imprima a média aritmética, com a mensagem **média** antes do resultado.

10. **Algoritmo 10**
    - Entre com dois números inteiros e imprima a seguinte saída:

      ```text
      Dividendo:
      Divisor:
      Quociente:
      Resto:
      ```
